SR (Scene Recon)

This project has many times at time, i wish this name was the last, the principal objective of the project is organize information on incident with a large work team. All investigators on project can contribute simultaneously with his own findings.

Primero hablemos de la infraestructura del proyecto, la cual consiste en una base de datos construida en mongodb, esta base de datos almacenara los objetos correspondientes a todos los registros realizados en la aplicacion la igual que usuarios y roles de trabajo. Por otra parte tenemos un backend construido en nodejs que se encargara tanto de la comunicacion entre el frontend y la base de datos como de todos los procesos de autenticacion y tratamiento de datos, finalmente el frontend construido en Angular (trato de mantenerlo siempre en la ultima version).

El deployment de la aplicacion esta disenado para ser realizado en docker, compuesto por tres maquinas independientes para cada una de las funciones mencionadas anteriormente. Proximamente realizare el primer post que explicara como hacer el deploy de la aplicacion y su primer uso.
